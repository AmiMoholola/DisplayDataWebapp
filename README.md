# Displaydata

C# .netcore mvc web application
Create a controller to display the Report view.
Create an action to read the BDG_Output.json file.(BDG_Output.json will be located in the C Drive, run the console Application(FileScanApp) to generate the file in the C drive)
Create a model that maps the JSON schema of the BDG_Output.json file.
Deserialize data to the model.
Link the model to the view.
Display the model data in a table using razor.
Screenshots provided of the out put in the screens Directory in the root of the project.
